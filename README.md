# Secure Blog Manager Ionic

Secure Blog Manager App is an app that works on a simple NodeJS CRUD Api to manage a Blog.

#**Technologies Used**

1. NodeJS
2. MongoDB
3. VueJS
4. AngularJS
5. Ionic Framework
6. Capacitor
7. Render
